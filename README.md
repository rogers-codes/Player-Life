# 🎵 Player Life

Player Life é uma aplicação desenvolvida em **Java com Spring Boot** e banco de dados **PostgreSQL** que permite cadastrar músicas, artistas e realizar buscas personalizadas.  
O objetivo é praticar conceitos de **APIs REST**, **persistência de dados** e **boas práticas no desenvolvimento backend**.

---

## 🛠️ Tecnologias Utilizadas

- [Java 17+](https://www.oracle.com/java/)
- [Spring Boot](https://spring.io/projects/spring-boot)
- [Spring Data JPA](https://spring.io/projects/spring-data-jpa)
- [PostgreSQL](https://www.postgresql.org/)
- [Maven](https://maven.apache.org/)

---

## 📌 Funcionalidades

- Cadastrar músicas 🎶  
- Cadastrar artistas 🎤  
- Listar músicas cadastradas  
- Buscar músicas por artista  
- Pesquisar dados de um artista
